## 投稿パターン分析

### ❶ 「断言型」（高エンゲージ安定）
- 「精神科医として言いたい」「ハッキリ言いますが」
- 「〇〇は△△ではない。□□だ。」
- → **権威性＋断定でクリック・保存を誘発**

### ❷ 「逆説・再定義型」（バズりやすい）
- 「メンタルが弱い人、実はこれなだけ」
- 「生きづらさは"できなさ"自体ではなく…」
- 「ADHDはわがままじゃない」
- → **常識を否定して本質を提示**

### ❸ 「カルーセル型」（1/2, 1/3）
- 上位投稿の多くがカルーセル（複数画像/ページ）
- 1枚目でフックを掛け、続きを見させる
- → **テキストでも「続きは↓」で長文を分割する効果**

### ❹ 「箇条書き型」（保存されやすい）
- ❶❷❸で構造化
- 「〇〇 ❌ → △△ ⭕️」のビフォーアフター形式
- → **一目で価値が分かる。スクショ・保存向き**

### ❺ 「当事者共感型」（フォロワー獲得）
- 「自分と同じように悩んでいる人の手段になりたい」
- 「同じ経験をしてるママと繋がりたい」
- → **共感でフォローを促す。長期成長向き**

## Threads vs note の違い

| 観点 | Threads | note |
|------|---------|------|
| 文字数 | 500文字以内が最適 | 1500〜2500字 |
| 構成 | フック→本質→一言で締め | 導入→展開×3-4→まとめ |
| 文体 | 断言・短文・体言止め | 語りかけ・体験談ベース |
| 装飾 | 絵文字OK・❶❷❸で構造化 | 見出し・太字・引用 |
| 目的 | 認知拡大・フォロワー獲得 | 深い共感・有料化 |
| ハッシュタグ | 3〜5個 | 3〜5個 |

## れんアカウントへの提言

### ポジショニング
- **精神科医でも専門家でもない「当事者の先輩」** → 権威ではなく共感で刺さる
- 東洋哲学×発達特性は **Threadsではほぼ未開拓**（競合ゼロに近い）
- noteの記事を要約してThreadsに流す **クロスポスト戦略** が有効

### 最適な投稿パターン（れん向け）
1. **当事者逆説型**: 「ADHDグレーの僕が〇〇をやめたら△△になった」
2. **東洋哲学×日常型**: 「老子の言葉で救われた話を30秒で。」
3. **箇条書きTips型**: 発達特性向けライフハックを❶❷❸で
4. **共感フック型**: 「これ、ADHDあるあるじゃないですか？」から始まる

### 投稿頻度の目安
- 毎日1投稿が理想（Threadsはアルゴリズム的に頻度が重要）
- note記事の要約版を1日1本 + オリジナル短文を混ぜる

## 次の投稿に使うべき公式

```
フック（共感 or 逆説）1行
→ 本質の再定義 2-3行
→ 具体的な対策/気づき（箇条書き❶❷❸）
→ 一言で締め（押し付けない）
→ ハッシュタグ 3-5個
```

**例:**
- 「ADHDグレーの僕が"努力"をやめた日から、仕事がうまくいき始めた。」
- 「禅の言葉が、2回うつになった僕を救った話。」
- 「"普通"になろうとして壊れた。発達グレーゾーンの僕が見つけた"自分の普通"。」
