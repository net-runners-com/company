# 経理部門

## 役割
売上・支出・請求書・税務関連の記録と管理を担当する。
個人事業主の財務状況を可視化し、確定申告や資金繰りをサポートする。

## 担当領域
- 売上・入金管理
- 経費・支出管理
- 請求書の発行・管理
- 月次・年次の収支サマリー
- 確定申告準備（経費分類など）

## フォルダ構成
- `invoices/` - 請求書管理（発行済み・入金待ち・入金済み）
- `expenses/` - 経費記録（月次ファイル）
- `reports/` - 月次・年次収支レポート

## ファイル命名規則
- 請求書: `invoice-YYYYMM-クライアント名.md`
- 経費: `expenses-YYYY-MM.md`
- レポート: `report-YYYY-MM.md` / `report-YYYY.md`

## 経費記録フォーマット
```markdown
| 日付 | 内容 | 金額 | 区分 | 備考 |
|------|------|------|------|------|
| YYYY-MM-DD | 内容 | ¥0,000 | 通信費/消耗品費/外注費/... | |
```

## 請求書フォーマット
```markdown
# 請求書 No.YYYYMM-XX

- 発行日: YYYY-MM-DD
- 支払期限: YYYY-MM-DD
- 請求先: クライアント名
- 件名: 作業内容
- 金額: ¥000,000（税込）
- ステータス: 発行済み / 入金待ち / 入金済み
```

## PDF帳票テンプレート

`templates/` にPython（fpdf2）ベースのPDF生成スクリプトがある。
帳票を作成する際は必ずこれらのテンプレートを使うこと。

| ファイル | 用途 | 実行方法 |
|---------|------|---------|
| `generate_invoice.py` | 請求書 | PARAMSを変更して `/usr/bin/python3` で実行 |
| `generate_estimate.py` | 見積書 | 同上 |
| `generate_contract.py` | 売買契約書 | 同上 |

### 共通仕様
- フォント: ヒラギノ角ゴシック W3/W6（macOS標準）
- 自社情報: NetRunners / 竹内大登 / 〒343-0011 埼玉県越谷市増林3500
- インボイス番号・口座番号等はPARAMS内で臨機応変に設定
- 消費税: 10%（税抜価格に対して計算）
- 請求書は源泉所得税の有無を `withholding_tax` で切替可能

### 使い方
各スクリプトの `PARAMS` 辞書を案件に合わせて書き換え、実行するとPDFが出力される。
`output_path` を指定しなければ `invoices/` に自動保存される。

## ルール
- 経費は発生したタイミングで記録する（後回し禁止）
- 請求書はステータスを必ず更新する
- 月末に収支サマリーを reports/ に作成する
- 個人事業主の主な経費区分: 通信費・消耗品費・外注費・広告宣伝費・研修費・水道光熱費
- **帳票PDF作成時は必ず `templates/` のスクリプトを使う**
