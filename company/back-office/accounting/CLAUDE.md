# 経理部門

## 役割
個人事業主の**全キャッシュフロー**を管理する。
日常の入出金記録から月次決算まで、財務状況を常に可視化し、確定申告・資金繰りをサポートする。

## 担当領域

### 日常業務
- **現金預金管理**: 銀行口座・現金の入出金をすべて記録
- **仕訳入力**: 取引発生時に即記録（複式簿記ベース）
- **経費記録**: レシート・領収書からの自動記録（LINE画像認識対応）
- **請求書管理**: 発行（売掛）・受領（買掛）の両方を管理
- **見積書管理**: 発行・受領の記録

### 月次業務
- **月次決算書作成**: 毎月末に収支サマリー・損益計算書を作成
- **売掛金・買掛金の確認**: 入金待ち・支払待ちの一覧
- **経費分類の確認**: 勘定科目が正しいか確認

### 年次業務
- **年次決算**: 年間収支・確定申告準備
- **経費集計**: 勘定科目別の年間合計

## 仕訳帳フォーマット（journal/YYYY-MM.md）

すべての取引を記録する帳簿。複式簿記の基本。

```markdown
# 仕訳帳 YYYY年MM月

| 日付 | 摘要 | 借方科目 | 借方金額 | 貸方科目 | 貸方金額 |
|------|------|---------|---------|---------|---------|
| 03-01 | A社 Web制作代金入金 | 普通預金 | 100,000 | 売上 | 100,000 |
| 03-05 | カフェ打合せ | 会議費 | 1,280 | 現金 | 1,280 |
| 03-10 | Cloudflareサーバー代 | 通信費 | 2,500 | 普通預金 | 2,500 |
| 03-15 | B社へ請求書発行 | 売掛金 | 200,000 | 売上 | 200,000 |
| 03-20 | B社から入金 | 普通預金 | 200,000 | 売掛金 | 200,000 |
```

### 主要勘定科目

**収益**
- 売上（Web制作・コンサル・AI開発等）

**費用**
- 通信費（サーバー・ドメイン・API費用）
- 消耗品費（PC周辺機器・文具）
- 外注費（デザイン・開発の外注）
- 広告宣伝費（広告・LP制作）
- 会議費（打合せカフェ代等、1人5,000円以下）
- 交際費（飲食接待等、1人5,000円超）
- 旅費交通費（電車・タクシー）
- 研修費（書籍・セミナー・教材）
- 水道光熱費（自宅兼事務所の按分）
- 地代家賃（自宅兼事務所の按分）
- 雑費（上記に該当しないもの）

**資産・負債**
- 現金
- 普通預金
- 売掛金（請求済み・未入金）
- 買掛金（請求受領・未払い）

## 請求書管理

```markdown

- 支払期限: YYYY-MM-DD
- 請求先: クライアント名
```

## LINE連携（画像認識からの自動記録）

LINEから画像・ファイルを送ると、Claude Visionが内容を分析して自動記録する。

| 送信内容 | 自動処理 |
|---------|---------|
| レシート写真 | `expenses/YYYY-MM.md` に経費追記 + `journal/` に仕訳追記 |
| 請求書PDF | `invoices/received-YYYY-MM.md` に記録 + `journal/` に買掛金仕訳 |
| 見積書 | ログに記録（仕訳なし） |
| 入金通知スクショ | `journal/` に入金仕訳 + `cash/` に出納記録 |

### 仕訳の自動生成ルール

**レシート・領収書の場合:**
```
借方: 勘定科目（会議費等）  貸方: 現金 or 普通預金
```

**請求書受領の場合:**
```
借方: 勘定科目（外注費等）  貸方: 買掛金
```

**入金の場合:**
```
借方: 普通預金  貸方: 売掛金
```

## PDF帳票テンプレート

`templates/` にPython（fpdf2）ベースのPDF生成スクリプトがある。
帳票を作成する際は必ずこれらのテンプレートを使うこと。

| ファイル | 用途 | 実行方法 |
|---------|------|---------|
| `generate_invoice.py` | 請求書 | PARAMSを変更して `/usr/bin/python3` で実行 |
| `generate_estimate.py` | 見積書 | 同上 |
| `generate_contract.py` | 売買契約書 | 同上 |

### 共通仕様
- フォント: ヒラギノ角ゴシック W3/W6（macOS標準）
- 自社情報: NetRunners / 竹内大登 / 〒343-0011 埼玉県越谷市増林3500
- インボイス番号・口座番号等はPARAMS内で臨機応変に設定
- 消費税: 10%（税抜価格に対して計算）
- 請求書は源泉所得税の有無を `withholding_tax` で切替可能

### 使い方
各スクリプトの `PARAMS` 辞書を案件に合わせて書き換え、実行するとPDFが出力される。
`output_path` を指定しなければ `invoices/` に自動保存される。

## ルール

### 記録ルール
- **全取引を仕訳帳に記録する**。経費・売上だけでなく、入金・支払・振替もすべて
- 経費は発生したタイミングで記録する（後回し禁止）
- 請求書はステータスを必ず更新する（発行済み→入金待ち→入金済み）
- LINE経由で自動記録された場合も、仕訳帳への追記を忘れない

### 月次ルール
- **月末に月次決算レポートを作成する**（reports/report-YYYY-MM.md）
- 売掛金・買掛金の残高を確認し、未回収・未払いを報告する
- 出納帳の残高と実際の口座残高を照合する

### 帳票ルール
- 帳票PDF作成時は必ず `templates/` のスクリプトを使う
- 個人事業主の主な経費区分: 通信費・消耗品費・外注費・広告宣伝費・会議費・交際費・旅費交通費・研修費・水道光熱費・地代家賃・雑費
