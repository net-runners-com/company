# AI Company - ページ構成 & 機能一覧

## ナビゲーション構成

サイドバー（fixed, 左 w-60）に10項目。モバイルはボトムナビ。
EN/JA 切り替えボタン付き。

| # | ナビキー | ルート | ページ名 |
|---|---------|--------|---------|
| 1 | dashboard | /home | ダッシュボード |
| 2 | analytics | /dashboard | 分析 |
| 3 | tasks | /tasks | タスク |
| 4 | schedule | /schedule | スケジュール |
| 5 | projects | /projects | プロジェクト |
| 6 | revenue | /revenue | 売上 |
| 7 | documents | /documents | 書類 |
| 8 | integrations | /sns | 連携 |
| 9 | activity | /activity | アクティビティ |
| 10 | settings | /settings | 設定 |

その他のルート: `/` (ランディング), `/onboarding`, `/employee/create`, `/employee/[id]`

## 各ページ機能詳細

### 1. ランディングページ (`/`)

- ロゴ + タグライン
- 「はじめる」→ `/onboarding`
- 「ログイン」→ `/home`

### 2. オンボーディング (`/onboarding`)

- 3ステップウィザード
  1. 会社情報（会社名、業種）
  2. ミッション（ミッション、目標）
  3. 完了画面（さくら紹介）
- ステップインジケーター
- Zustand で会社情報を保存

### 3. ダッシュボード (`/home`)

- 挨拶ヘッダー（時間帯で変化）
- 統計カード4枚（社員数、完了タスク、進行中、総タスク）
- **部門別チームメンバー一覧**（部門ごとにグループ化）
  - 社員カード: アバター(react-nice-avatar) + 名前 + ロール + ステータス + 部署 + スキル + 挨拶
- 最近のアクティビティ（右カラム、5件）

### 4. 分析 (`/dashboard`)

- 統計カード4枚（社員数、完了タスク、進行中、SNS投稿）
- **アクティビティ推移チャート**（SVGラインチャート、週/月/年切替）
- 最近のイベント（右カラム、タイプ別カラードット）

### 5. タスク (`/tasks`)

- サマリーカード3枚（未着手、進行中、完了）
- フィルタータブ（すべて / 未着手 / 進行中 / 完了）
- タスクリスト
  - ステータスドット + タイトル + 説明
  - 優先度バッジ（高/中/低）
  - ステータスバッジ
  - 期限日
  - 担当者アバター
- モックデータ7件

### 6. スケジュール (`/schedule`)

- **カレンダー/リスト切り替えトグル**
- **カレンダービュー**（Google Calendar風）
  - 月表示グリッド（6週 x 7日）
  - 前月/翌月ナビゲーション + 今日ボタン
  - 曜日カラー（日=赤、土=青）
  - 今日はパープル丸で強調
  - セル内にイベントバッジ（タイプ別色）
  - 日付クリック → 右パネルにイベント詳細
- **リストビュー**
  - 日付ごとにグループ化
  - イベントカード（左ボーダー色=タイプ別）
  - 時間 + タイトル + タイプバッジ + メンバーアバター
- モックデータ6件（MTG、締切、レビュー等）

### 7. プロジェクト (`/projects`)

- サマリーカード3枚（進行中数、予算合計、使用済合計）
- フィルタータブ（すべて / 進行中 / 完了 / 保留）
- プロジェクトカード（2カラムグリッド）
  - 名前 + クライアント + ステータスバッジ
  - 説明（2行クランプ）
  - **予算消化プログレスバー**（色分け: 緑→黄→赤）
  - メンバーアバター（重ね表示）
  - 開始日〜終了日
- モックデータ4件

### 8. 売上 (`/revenue`)

- サマリーカード4枚（総売上、月平均、今月、成長率%）
- **週/月/年 切り替えタブ**
- **コンボチャート**（バー + ライン、SVG描画）
  - バー: 薄いパープル背景
  - ライン: パープル実線 + ドット
  - X軸ラベル（曜日/週/月、日英対応）
- データテーブル（期間ごとの金額一覧）
- 金額は `Intl.NumberFormat` で JPY 通貨フォーマット

### 9. 書類 (`/documents`)

- サマリーカード4枚（見積書数、請求書数、入金済、未入金）
- フィルタータブ（すべて / 見積書 / 請求書）
- **テーブル表示**
  - 番号 + タイプバッジ（見積書=青、請求書=紫）
  - クライアント名
  - 件名（md以上で表示）
  - 金額（右寄せ、太字）
  - ステータスバッジ（下書き/送付済/入金済/期限超過/キャンセル）
  - 発行日（lg以上で表示）
- モックデータ4件

### 10. SNS連携 (`/sns`)

- 「アカウントを連携」ボタン
- アカウントリスト
  - プラットフォームアイコン + 名前
  - アカウント名
  - ステータスバッジ（接続中 / 再接続）
  - メニューボタン
- 対応プラットフォーム: note, Threads, LINE, X, Instagram
- モックデータ2件（note, Threads）

### 11. アクティビティ (`/activity`)

- アクティビティリスト
  - タイプ別カラードット（チャット=青、タスク=紫、投稿=緑、エラー=赤、システム=灰）
  - サマリーテキスト
  - タイプラベル + 経過時間
- モックデータ5件

### 12. 設定 (`/settings`)

- **会社情報**（会社名、業種、ミッション、目標）
  - 編集フォーム + 保存ボタン
- **プラン**（Free / Upgrade ボタン）
- **データ**（エクスポートボタン）

### 13. 社員作成 (`/employee/create`)

- 2ステップウィザード
  1. 基本情報（名前、役割セレクト、部署）
  2. パーソナリティ & スキル（口調セレクト、スキルタグ追加/削除）
- 役割: Secretary, Writer, Researcher, Sales, Accounting, Designer, Engineer
- 口調: Polite, Friendly, Intellectual, Energetic, Cool, Gentle

### 14. 社員詳細 (`/employee/[id]`)

- ヘッダー: アバター + 名前 + ロール + ステータスバッジ
- **3タブ切り替え**
  - **チャット**: LINE風バブルUI、入力フィールド、ローディングアニメーション
  - **タスク**: 担当タスク一覧、ステータスバッジ
  - **設定**: 名前/ロール/口調 編集フォーム、スキルタグ表示

## 共通機能

### 多言語対応 (i18n)
- 日本語 / 英語 切り替え
- サイドバー下部に EN/JA ボタン
- localStorage に保存（リロード後も維持）
- ブラウザ言語で初期値自動判定
- 全ページ・全コンポーネント対応

### デザインシステム
- カラー: パープル (#7C5CFC) アクセント、白背景
- フォント: Inter
- 全カードにソフトシャドウ + ホバーで浮く
- パープルボタンに色付きシャドウ
- サイドバー fixed + シャドウ
- レスポンシブ: PC サイドバー / モバイル ボトムナビ

### アバター
- react-nice-avatar 使用
- employee.id をシードに決定的生成（同じ社員=同じ顔）
- 社員カード、チャット、詳細ヘッダー、プロジェクトメンバーで使用

### データレイヤー
- 現在: モックデータ (`src/data/mock.ts`)
- API 抽象レイヤー (`src/lib/api.ts`) — 関数シグネチャ固定
- 将来: PostgreSQL (worker経由) に差し替え予定
- Zustand ストア: company, employees, chat
