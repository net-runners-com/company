# FocusDrop — ADHD向けTODOアプリ 設計書

作成日: 2026-03-26
ステータス: 実装中

---

## コンセプト
「やることが多すぎて動けない」ADHDの人が**今やるべき1つ**に集中できるTODOアプリ。
ドーパミンを刺激する報酬演出で「完了の快感」を最大化する。

## ADHD向け設計原則
1. **1タスク集中**: 一度に1つだけ表示する「フォーカスモード」
2. **脳内ダンプ**: 思いついたら即メモ。分類は後回し
3. **小さく分解**: タスクにサブタスク（ステップ）を持たせる
4. **時間の可視化**: ポモドーロタイマーで25分だけ集中
5. **報酬**: 完了時のアニメーション・連続記録・レベルアップ
6. **過負荷防止**: 1日のタスク上限を設定（デフォルト5個）

## 画面構成

```
Tab Navigation
├── 🎯 フォーカス (Focus)     — 今やる1タスク + ポモドーロタイマー
├── 📥 ダンプ (Dump)          — 思いついたら即入力。脳を空にする
├── 📋 タスク (Tasks)         — 全タスク一覧 + フィルタ
└── 🏆 記録 (Stats)           — 連続日数・完了数・レベル
```

## 技術
- Expo SDK 54 + TypeScript
- expo-router (タブナビゲーション)
- expo-haptics (触覚フィードバック)
- expo-linear-gradient (グラデーション)
- AsyncStorage (ローカル永続化)
- react-native-reanimated (アニメーション)
