### はじめに

「AIにコードを書かせる」と聞いて、どんなイメージがありますか？

ChatGPTにコードを貼り付けて修正してもらう——そんな使い方に慣れている方も多いと思います。でも **Claude Code** は、それとは次元が違います。

プロジェクト全体を読んで、ファイルを横断して編集して、コマンドまで実行してくれる。まるで優秀なエンジニアが隣にいるような感覚です。

この記事では、Claude Code を触ったことがない方向けに **「何ができるのか」「どう始めるのか」** をわかりやすく解説します。

---

### Claude Code とは？

Claude Code は Anthropic が開発した **エージェント型のAIコーディングツール**です。

普通のAIチャットと何が違うかというと——

| 普通のAIチャット | Claude Code |
|-----------------|-------------|
| コードを貼って相談 | プロジェクト全体を自動で読む |
| 1ファイルずつ対応 | 複数ファイルを横断して編集 |
| 提案するだけ | コマンドも実行してくれる |
| セッションが消える | CLAUDE.md に記憶を保存 |

ターミナルで動くCLIが基本ですが、VS Code、JetBrains、デスクトップアプリ、ブラウザからも使えます。

---

### インストール方法（3分で完了）

**macOS / Linux:**
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows PowerShell:**
```powershell
irm https://claude.ai/install.ps1 | iex
```

インストールが終わったら、プロジェクトのフォルダで起動するだけです。

```bash
cd あなたのプロジェクト
claude
```

ブラウザで認証が求められるので、Claudeアカウントでログインすれば完了。

> ⚠️ 注意: 無料プランでは使えません。Claude Pro（月$20）以上が必要です。

---

### 最初にやること（5分で体験できる）

起動したら、まずこれを打ってみましょう。

```
what does this project do?
```

これだけで、プロジェクト全体を読んで構造を説明してくれます。

次に実際の編集を体験するなら：

```
add a hello world function to the main file
```

変更内容を確認してから承認できるので、安心して試せます。

### 実際に何ができるのか

#### ① バグ修正
```
I'm seeing this error when I run npm test
[エラーメッセージを貼り付ける]
```
エラーの根本原因を特定して修正してくれます。テストも自動実行。

#### ② 機能追加
```
ユーザー登録フォームにバリデーションを追加して
```
関連するファイルを自分で探して、実装まで完了させてくれます。

#### ③ Git操作
```
変更をコミットして、わかりやすいメッセージをつけて
```
差分を読んで適切なコミットメッセージを生成。PRの作成もできます。

#### ④ コードの説明・リファクタリング
```
utils.jsを現代的なJavaScriptに書き直して
```
既存のスタイルを学習した上でリファクタリングしてくれます。

### 覚えておくと便利なコマンド

| コマンド | 内容 |
|---------|------|
| `/help` | コマンド一覧を表示 |
| `/plan` | 実行前に計画だけ確認（読み取り専用） |
| `/diff` | 変更差分をインタラクティブに確認 |
| `/cost` | トークン使用量・コストを確認 |
| `/memory` | CLAUDE.md（永続メモ）を編集 |
| `/clear` | 会話履歴をリセット |

---

### CLAUDE.md でプロジェクトを記憶させる

プロジェクトのルートに置いておくと、毎回読み込んでくれます。

```markdown
# プロジェクトルール

## コードスタイル
- async/awaitを使う
- コンポーネントは200行以内に

## テスト
- Jest でユニットテスト
- 変更後は必ずテスト実行
```
