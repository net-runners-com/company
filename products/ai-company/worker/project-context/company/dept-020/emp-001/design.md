## 2. 使用技術

| 項目 | 技術 | 選定理由 |
|------|------|----------|
| マークアップ | HTML5 | 標準・SEO対応 |
| スタイル | CSS3 + Tailwind CSS (CDN) | 高速開発・レスポンシブ対応が容易 |
| JavaScript | Vanilla JS (ES6+) | 予算内で軽量に。フレームワーク不要 |
| フォーム | Googleフォーム or Formspree | 無料・サーバーレス |
| ホスティング | 静的サイト（納品） | クライアント側で任意のサーバーに設置可能 |
| アイコン | Font Awesome (CDN) | 無料・種類豊富 |
| フォント | Google Fonts（Noto Sans JP） | 日本語対応・無料 |

### 選定方針
- 予算20万円のため、WordPress等のCMSは使わず**静的HTML**で制作
- クライアントが更新しやすいよう、ニュース・実績はシンプルなHTML構造で作成
- 将来的にCMS化したい場合は追加提案可能

---

## 3. レスポンシブ対応方針

### ブレークポイント

| デバイス | 幅 | 対応 |
|----------|-----|------|
| モバイル | ~767px | シングルカラム、ハンバーガーメニュー |
| タブレット | 768px~1023px | 2カラム、ナビゲーション調整 |
| デスクトップ | 1024px~ | フルレイアウト、最大幅1280px |

### 方針
- **モバイルファースト**で設計（建築業界はスマホからの問い合わせが多い）
- 画像は`srcset`で複数解像度対応、WebP形式を優先
- タップターゲットは最低44px×44px確保
- 電話番号はスマホでタップ発信対応（`tel:`リンク）

## 6. 共通UI仕様

### ヘッダー
- スクロール時に背景白+影で固定表示
- モバイル時：ハンバーガーメニュー（スライドイン）
- 電話番号はPC時のみ表示、モバイルは電話アイコンタップ

### フッター
- 3カラム構成（会社情報 / サイトマップ / SNS・問い合わせ）
- モバイル時：1カラム縦積み

### パンくずリスト
- トップ以外の全ページに設置
- 構造化データ（JSON-LD）対応でSEO強化

### カラー方針（提案）
- メインカラー: 紺系（#1a365d）— 信頼感・安定感
- アクセント: オレンジ系（#dd6b20）— 活力・CTAボタン
- 背景: 白（#ffffff）+ 薄グレー（#f7fafc）セクション交互
- テキスト: 濃いグレー（#2d3748）

### フォント
- 見出し: Noto Sans JP Bold (700)
- 本文: Noto Sans JP Regular (400)
- 基本サイズ: 16px / 行間: 1.75

## 7. SEO・パフォーマンス対策

- 各ページに適切な`<title>`と`<meta description>`設定
- 構造化データ（JSON-LD）: Organization, BreadcrumbList, LocalBusiness
- OGP設定（SNSシェア対応）
- 画像の遅延読み込み（`loading="lazy"`）
- CSS/JSの最小化
- Core Web Vitals を意識したレイアウト（CLS対策）
