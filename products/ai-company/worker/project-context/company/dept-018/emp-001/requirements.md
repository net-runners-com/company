## 2. ページ構成（想定）

予算20万円の範囲で、以下の5〜7ページ構成を想定。

| No. | ページ | 内容 | 優先度 |
|-----|--------|------|--------|
| 1 | トップページ | メインビジュアル、事業概要、施工事例ピックアップ、お知らせ、CTAボタン | 必須 |
| 2 | 会社概要 | 会社名、代表者、所在地、設立年、資格・許認可、沿革、アクセスマップ | 必須 |
| 3 | 事業内容 | 対応工事の種類（新築・リフォーム・商業施設等）、強み・特徴 | 必須 |
| 4 | 施工事例 | 写真ギャラリー形式、案件ごとの説明（工期・規模等） | 必須 |
| 5 | お問い合わせ | フォーム（名前・電話・メール・相談内容）、電話番号掲載 | 必須 |
| 6 | 採用情報 | 募集職種、待遇、応募方法 | 任意 |
| 7 | お知らせ/ブログ | 新着情報、施工レポート等 | 任意 |

## 3. ヒアリング項目（確認が必要な事項）

### 3-1. 基本情報
- [ ] 会社正式名称・代表者名・設立年
- [ ] 所在地・電話番号・FAX・メールアドレス
- [ ] 事業内容の詳細（対応工事の種類）
- [ ] 許認可・資格情報（建設業許可番号等）
- [ ] 既存のホームページの有無（リニューアルか新規か）
- [ ] 独自ドメインの有無（希望ドメイン）

### 3-2. デザインの方向性
- [ ] 希望するイメージ（信頼感・モダン・親しみやすい等）
- [ ] コーポレートカラー / ロゴデータの有無
- [ ] 参考にしたい他社サイトのURL
- [ ] ターゲット層（個人住宅向け / 法人向け / 両方）

### 3-3. 素材関連
- [ ] 施工事例の写真素材の有無（何件分あるか）
- [ ] 代表者・スタッフの写真素材
- [ ] 掲載テキスト（会社紹介文・挨拶文等）の原稿の有無
- [ ] ロゴデータ（AI/PNG等）の有無

### 3-4. 機能要件
- [ ] お問い合わせフォームの必要項目
- [ ] Googleマップ埋め込みの要否
- [ ] SNS連携（Instagram・Facebook等）の要否
- [ ] スマートフォン対応（レスポンシブ）→ 必須として提案
- [ ] SSL対応（https）→ 必須として提案
- [ ] アクセス解析（Google Analytics）導入の要否

### 3-5. 運用・スケジュール
- [ ] 公開希望日
- [ ] 公開後の更新頻度・更新担当者
- [ ] CMS（WordPress等）の必要性
- [ ] サーバー・ドメインの契約状況
- [ ] 保守・運用サポートの希望有無（別途見積）

## 4. 予算内での制作方針（提案）

### 20万円で対応可能な範囲
- テンプレートベースのデザイン（WordPress or 静的サイト）
- レスポンシブ対応（PC・スマホ・タブレット）
- 5ページ程度の制作
- お問い合わせフォーム設置
- 基本的なSEO対策（meta情報、構造化データ）
- SSL対応
- Google Analytics設置

### 予算外（オプション・追加費用が必要）
- オリジナルデザイン（フルスクラッチ）
- 撮影代行（施工現場・スタッフ写真）
- ライティング代行（原稿作成）
- 月額保守・更新サポート
- 追加ページ（6ページ目以降）

## 7. 備考

- 予算20万円は建築会社のコーポレートサイトとしてはコンパクトな予算帯。テンプレート活用で品質とコストのバランスを取る方針。
- 施工事例の写真が充実しているかが、サイトの仕上がりを大きく左右するため、素材確認を最優先で進める。
- CMS導入の場合はWordPressを想定。お客様側での更新が不要ならば静的サイトも選択肢（コスト削減可能）。
